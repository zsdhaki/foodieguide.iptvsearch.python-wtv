# foodieguide.iptvsearch.python-wtv
为https://www.foodieguide.com/iptvsearch 做的适配wtv导入的python爬虫

## 安装
下载release里最新文件[release]
这个python项目依赖requests，re，bs4库，使用以下命令安装

```sh
pip install requests
pip install re
pip install bs4
```

## 使用
Windows终端执行
```sh
python iptv-research.py
```
